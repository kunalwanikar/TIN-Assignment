==================================================Instruction to use Code of Problem 2 ====================================================
Compile the code using the command "g++ problem2.cpp -o problem2"

After compiling successfully you can run the code using the command "./problem2"

Now follow the below steps to execute the code.

1) Enter the value of Lambda, mu and total time units you want to observe the queue.

2) After that you will get a option to select a case which you want to perform.
   Press 1 and then enter for Case A
   Press 2 and then enter for Case B
   Press 3 and then enter for Case C

3) You will get the desired output on the console as well as a file will be created for each case separately which will store the output for every case.
